**Fast & Easy Custom Code Snippets for WordPress**  
Add unlimited custom CSS, JavaScript, and HTML snippets to your WordPress website with ease. Performance-optimized, file-based, secure, beginner-friendly, and AI-assisted.

Simple Custom Code is the most intuitive and powerful WordPress plugin for adding custom CSS, JavaScript, and HTML code to your website. Whether you’re a beginner or an experienced developer, this plugin makes it incredibly easy to customize your WordPress site without touching theme files or risking data loss. Unlike other custom code plugins that store snippets in your database (slowing down your site), Simple Custom Code stores all your code in optimized files, ensuring lightning-fast performance and better security.

## Installation
1. Download the latest release from [Releases](https://github.com/appscreo/simple-custom-code/releases).
2. Upload the ZIP via **WordPress Admin > Plugins > Add New > Upload Plugin**.
3. Activate the plugin.
4. *(Optional)* Add your OpenAI API key in **Settings > Simple Custom Code** for AI features.
5. Start adding snippets!

**Pro Users**: Purchase a license at [simplecustomcode.com/pricing](https://simplecustomcode.com/pricing/), then enter your key in the plugin settings to unlock advanced features.
